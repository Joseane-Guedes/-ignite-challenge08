# 🔧 Run Locally

Clone the project

```bash
  git clone git@github.com:git@github.com:Joseane-Guedes/Ignite-challenge08.git
```

Go to the project directory

```bash
  cd Ignite-challenge08
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
